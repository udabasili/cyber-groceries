# CyberGroceries
## eCommerce app



CyberGroceries is a mobile-ready, user-friendly, e-commerce website that also possesses an administrator section.



## Features

- Fully functional Administrator section which is accessed with email user@administrator.ca and password 123456789
- Administration panel that allows users to control users, products, and orders
- Graphic interface for administrator showing sales and total changes in new users
- Ability to add, delete products from carts with real-time changes
- Full Authentication that makes use of tokens for user verification
- Shipping and delivery options
